🔧 Integration Guide — All 6 Sites

Copy the three lines below into the <head> and one line at the top of <body> of each website. The JS auto-detects which site it's on and highlights the correct button. 

<!-- Sir Ganguly Unified Navigation — paste in <head> -->
<link rel="stylesheet" href="https://sirganguly.com/sirganguly-nav.css">
<script src="https://sirganguly.com/sirganguly-nav.js" defer></script>

<!-- Paste this as the FIRST child of <body> -->
<div id="sg-nav-root"></div>

📋 Technical Notes

📐

Breakpoints

≥ 1024px — full desktop row
640–1023px — compact tablet row
< 640px — hamburger + drawer

⚡

Performance

CSS ~5 KB · JS ~4 KB (uncompressed).
Zero external dependencies.
Single HTTP round-trip to sirganguly.com CDN.

🔍

SEO

All links are real <a href> tags, fully crawlable by Google and Bing without JavaScript.

🎨

Theming

All colours controlled via CSS custom properties in :root — override any gradient per-site if needed.

♿

Accessibility

aria-current="page", aria-expanded, aria-label, Escape key closes drawer, visible focus rings.

📦

Public API

Access via window.SirGangulyNav — exposes .refresh(), .showProgress(), .hideProgress().
